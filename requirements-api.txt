# --- PyTorch ---
# NOT pinned here deliberately. The Dockerfile's base image
# (pytorch/pytorch:2.5.1-cuda12.1-cudnn9-runtime) already bundles a
# CUDA-matched torch build. Installing torch again via pip risks pulling
# the default CPU-only wheel from PyPI and silently breaking GPU support.
# If running this requirements file OUTSIDE that Docker image (e.g. bare
# venv), install torch separately first via the official CUDA index:
#   pip install torch --index-url https://download.pytorch.org/whl/cu121
